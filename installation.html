Installation

node.js

little-spas runs under node.js You can download it from nodejs.org

redis

little-spas uses the redis key-value store available from redis.io

little-spas

little-spas is a standalone node.js application. It is not meant to be used as a module in another application. It should be installed globally with NPM.

Global install using NPM:

First install the application

$ [sudo] npm install little-spas -g

Create a Project

You will need a folder for your little-spas project. It can be anywhere you like

$ cd /var/foo
$ mkdir myProject
$ cd myProject

Next use little-spas and NPM to add the necessary files to your project

$ little-spas --create
$ npm install

Test

At this point you can test little-spas. A sample bundle configuration is included for testing. Launch little-spas from your project directory: 

node little-spas --dev

From your browser navigate to http://localhost:3000/bundle/sample. You should get a JSON object in the response.

Configure

little-spas uses a basic configuration file (config.json). You'll want to edit this file to match your environment. There are two branches to the config file: development and live. If you pass the dev parameter to little spas: 

node little-spas --dev

the development values will be used. Otherwise, the live values will be used.

Here's a sample config file:

{
  "development": {
    "url": "http://localhost:3000",
    "port": 3000,
    "redis": {
      "port": 6379,
      "address": "localhost"
    }		
  },
  "live": {
    "url": "http://spas.mycompany.com",
    "port": 80,
    "redis": {
      "port": 6379,
      "address": "nodejitsudb123456789.redis.irstack.com",
      "auth": "nodejitsudb123456789.redis.irstack.com:f123abc456d978901e23f4g567hijkl8"
    }
  }
}

url: The root URL for little-spas
port: This is the port on which little-spas will listen for requests
redis: The redis configuration object
redis.port: The port that redis is listening on
redis.address: The address of your redis server
reds.auth: (Optional) If your redis server is password protected, use the password here
